# Brain Games

Консольный проект на PHP с набором математических мини-игр:
- `brain-even`
- `brain-calc`
- `brain-gcd`
- `brain-progression`
- `brain-prime`

## Минимальные требования

- PHP 8.1 и выше
- Composer 2 и выше
- GNU Make
- Git

## Установка

```bash
git clone https://github.com/ewqds-code/php-project-45.git
cd php-project-45
make install
```

## Запуск

```bash
make brain-games
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```

Также можно запускать напрямую:

```bash
./bin/brain-games
./bin/brain-even
./bin/brain-calc
./bin/brain-gcd
./bin/brain-progression
./bin/brain-prime
```
