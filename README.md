# Brain Games

`Brain Games` — набор из 5 консольных математических игр.  
Проект выполнен в рамках курса Hexlet по PHP.

## Минимальные требования

- PHP 8.1+
- Composer 2+
- Git
- Linux / macOS / WSL (для Windows можно приложить скриншоты вместо asciinema)

## Установка

```bash
git clone https://github.com/ewqds-code/php-project-45.git
cd php-project-45
make install
```

## Запуск игр

```bash
./bin/brain-games
./bin/brain-even
./bin/brain-calc
./bin/brain-gcd
./bin/brain-progression
./bin/brain-prime
```
